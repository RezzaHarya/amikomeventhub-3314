## 📌 Deskripsi Proyek
**AmikomEventHub** adalah platform manajemen event yang dikembangkan untuk mempermudah koordinasi dan publikasi acara di lingkungan kampus. Proyek ini dibangun menggunakan framework **Laravel 12** sebagai bagian dari tugas akademik di Universitas Amikom Yogyakarta.

## 🚀 Fitur Utama
* **Dashboard Admin:** Manajemen data event secara real-time.
* **Sistem CRUD:** Pengelolaan data mahasiswa dan kategori event yang sistematis.
* **Responsive UI:** Tampilan yang optimal di perangkat desktop maupun mobile.
* **Cloud Deployment:** Sudah terintegrasi dengan Laravel Cloud.

## 🛠️ Tech Stack
* **Framework:** [Laravel 12](https://laravel.com)
* **Database:** MySQL / MariaDB
* **Styling:** Tailwind CSS / Bootstrap (sesuaikan)
* **Deployment:** Laravel Cloud
